Principles

  • Everything should be 100% responsive
  • Everything should be readable on any device
  • Everything should be as performant as possible
  • Designing in the browser should be easy
  • Doing one thing extremely well promotes reusability

Tachyons on NPM

Tachyons is also available as a series of modules on npm and github that you can mix and match to suit your project's needs.

npm install --save-dev tachyons-module-name

A tachyon /ˈtæki.ɒn/ or tachyonic particle is a hypothetical particle that always moves faster than light.
The word comes from the Greek: ταχύς or tachys, meaning "swift, quick, fast, rapid"
